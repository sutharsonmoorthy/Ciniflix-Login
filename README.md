# 🎬 CINIFLIX - Cinematic Experience

A high-performance, responsive full-stack movie streaming dashboard built with the **MERN** stack (React, Node.js, Express) and styled with **Tailwind CSS**.

## ✨ Features
* **Authentication:** Secure login system using a Mock User database and Axios.
* **Modern UI:** Responsive Netflix-inspired design with a dark aesthetic.
* **Dynamic Interactivity:** Smooth scrolling headers, hover effects, and modal-like movie cards.
* **Mobile First:** Fully optimized for all screen sizes (Mobile, Tablet, Desktop).
* **Glassmorphism:** Elegant UI elements using Tailwind's backdrop-blur and opacity layers.

## 🛠️ Tech Stack

### Frontend
* **React 19** - UI Component logic.
* **Tailwind CSS** - Styling and responsive design.
* **React Router Dom** - Client-side navigation.
* **Axios** - API communication.

### Backend
* **Node.js & Express** - Server-side logic and API routing.
* **CORS** - Cross-Origin Resource Sharing for secure frontend-backend connection.

## ⚙️ Local Setup

1. Clone the repository:
    git clone (https://github.com/sutharsonmoorthy/ciniflix.git)
    cd ciniflix

2. Setup Backend:
    cd backend
    npm install
    npm start

3. Setup Frontend:
    cd ../frontend
    npm install
   
    Create a .env file and add VITE_API_URL=http://localhost:5000
    npm run dev
